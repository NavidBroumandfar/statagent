# Figures Directory

This directory contains visualizations generated by StatAgent examples and analyses.

## Generated Figures

When you run the example scripts, the following figures will be created:

- `negative_binomial_meteorites.png` - PMF of meteorite count distribution
- `survival_mixture_pdf.png` - PDF of survival mixture model
- `survival_mixture_histogram.png` - Histogram of simulated waiting times
- `polynomial_regression_comparison.png` - OLS vs Ridge regression comparison

## Using Your Own Output

You can specify custom output paths in all plotting functions:

```python
model.plot_pmf(save_path="figures/my_custom_plot.png")
```

## Figure Formats

All figures are saved as PNG with 300 DPI by default, suitable for:
- Publications
- Presentations
- Web display
- Reports

To change format or resolution, modify the save calls:

```python
plt.savefig("output.pdf", dpi=600, format='pdf')  # High-res PDF
plt.savefig("output.png", dpi=150, format='png')  # Web-optimized PNG
```

## Best Practices

1. **Descriptive names**: Use clear, descriptive filenames
2. **Organization**: Group related figures in subdirectories
3. **Version control**: Generally don't commit generated figures to git
4. **Documentation**: Note figure generation code in notebooks/scripts
